2. Instalación

1. Copiar la carpeta "WEB" en el directorio raíz del servidor. Si al acceder a la dirección del servidor no eres redirigido a index.php, configura tu servidor para que url_servidor/WEB sea a donde se les redirige a los usuarios.
2. Configurar base de datos

- Fichero: WEB/database/connect-database.php
$servername = "localhost";
$username = "Xelopez109";
$password = "";
$database = "Xelopez109_serieously";
Los campos $servername, $username y $password se deberán de configurar con la información de nuestro servidor de mysql.

- Archivo "backup.sql" o líneas siguientes para la recuperación de la base de datos:

Si quieres importar el archivo "backup.sql", crea una base de datos de nombre "Xelopez109_serieously" (CREATE DATABASE Xelopez109_serieously;) e importa el archivo (mysql -u username -p Xelopez109_serieously < ruta/backup.sql)
